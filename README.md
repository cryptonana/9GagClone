# 9GagClone

<h2> Building a 9gag-like forum </h2>
<h3>with:</h3>

- Profile System

- Posts System

- Comment System

# To download this repo

<code>git pull </code> this repo

<code>composer update</code>

Create a db and register the db with the config file

<code>php artisan migrate</code>
